TypeScript

JavaScript that scales

Intalling

  • Via npm (the Node.js package manager)
  • By installing TypeScript’s Visual Studio plugins
**For NPM users:** ```javascript > npm install -g typescript ```
**Create index.ts** ```javascript function greeter(person: string) { return "Hello, " + person; } let user = "Jane User"; document.body.textContent = greeter(user); ```
**At the command line, run the TypeScript compiler:** ```javascript tsc index.ts ``` *The result will be a file index.js*

Basic data types of TypeScript:

  • Number Types
  • String Types
  • Boolean Types
  • Null Types
  • Undefined Types
  • Array Types

**Number Types:** ```javascript let decimalValue: number = 10; ```

**String Types:** ```javascript let firstName: string = "Kate"; ```

**Boolean Types:** ```javascript let isPrimaryAccount: boolean = true; ```

**Null Types:** ```javascript let nullValue: null = null; ```

**Underfined Types:** ```javascript let undefinedValue: undefined = undefined; ```

*Null and undefined use more common to define types* ```javascript type SomeType = string | undefined | null; ```

**Array Types:** ```javascript let marks: number[] = [80, 85, 75]; ```

**Generic way to declare arrays:**

```javascript let marks: Array< number > = [80, 85, 75]; ```

Basic data types of TypeScript:

  • Tuple Types
  • Any Types
  • Void Types
  • Never Types
  • Enum Types

**Tuple Types:** ```javascript // correct let person: [string, number] = ["Kate", 1990]; ```

```javascript // error let person: [string, number] = [1990, "Kate"]; ```

**Any Types:** ```javascript let dynamicValue: any = "Kate Yakubouskaya"; dynamicValue = 1990; dynamicValue = true; ```

**Void Types:** ```javascript function showMessage(): void { console.log("Hello World"); } ```

**Never Types:** ```javascript function throwError(message: string): never { throw new Error(message); } ```

Enum Types:

Enums allow us to define a set of named constants

**Numeric enum:** ```javascript enum Direction { Up = 1, Down, Left, Right, } ```

String enum:

In a string enum, each member has to be constant-initialized with a string literal.

```javascript enum Direction { Up = "UP", Down = "DOWN", Left = "LEFT", Right = "RIGHT", } ```

```javascript interface Client { id:string age: number name: { firstName: string secondName: string } } ```

```javascript const client1: Client = { id: '23456', age: 29, name: { firstName: Kate, secondName: Yakubouskaya } } ```

Optional Properties

```javascript interface Client { id:string age?: number name: { firstName: string secondName: string } } ```

```javascript const client1: Client = { id: '23456', name: { firstName: Kate, secondName: Yakubouskaya } } client1.age = 29 // correct ```

Functions

**Typing the function** ```javascript function add(x: number, y: number): number { return x + y; } let myAdd = function(x: number, y: number): number { return x + y} ```

Default Parameters

```javascript function buildName(firstName: string, lastName: string) { return firstName + " " + lastName; } ```

```javascript let result1 = buildName("Bob"); // error let result2 = buildName("Bob", "Ady", "Sr."); // error let result3 = buildName("Bob", "Ady"); // ah, just right ```

Optional Parameters

```javascript function buildName(firstName: string, lastName?: string) { if (lastName) return firstName + " " + lastName; else return firstName; } ```

```javascript let result1 = buildName("Bob"); // works correctly let result2 = buildName("Bob", "Ady", "Sr.");// error let result3 = buildName("Bob", "Ady"); // ah, just right ```

Classes

```javascript class Greeter { greeting: string; constructor(message: string) { this.greeting = message; } greet() { return "Hello, " + this.greeting; } } ```

```javascript let greeter = new Greeter("world"); ```

Public modifiers

We have freely access the members that we declared throughout our programs.In TypeScript, each member is public

```javascript class Animal { name: string; constructor(theName: string) { this.name = theName; } } ```

**The same:** ```javascript class Animal { public name: string; public constructor(theName: string) { this.name = theName; } } ```

Private modifiers

When a member is marked private, it cannot be accessed from outside of its containing class.

```javascript class Animal { private name: string; constructor(theName: string) { this.name = theName; } } ```

```javascript new Animal("Cat").name; // Error: 'name' is private; ```

Protected modifiers

The protected modifier acts much like the private modifier with the exception that members declared protected can also be accessed within deriving classes.

```javascript class Person { protected name: string; constructor(name: string) { this.name = name; } } class Employee extends Person { constructor(name: string) { super(name); } } ```

```javascript let howard = new Employee("Howard"); console.log(howard.name); // error ```
